# Создание VK бота

## Шаг 1: Создайте сообщество

1. Откройте [vk.com/groups](https://vk.com/groups)
2. Нажмите "Создать сообщество"
3. Выберите тип (например, "Бизнес")
4. Введите название и создайте

## Шаг 2: Включите сообщения сообщества

1. Перейдите в "Управление" → "Сообщения"
2. Включите "Сообщения сообщества"
3. Включите "Возможности ботов"

## Шаг 3: Получите токен доступа

1. Перейдите в "Управление" → "Настройки" → "Работа с API"
2. Нажмите "Создать ключ"
3. Выберите права:
   - Управление сообществом
   - Сообщения сообщества
4. Нажмите "Создать"

Токен будет выглядеть так:
```
vk1.a.XXXXX...очень_длинная_строка...XXXXX
```

## Шаг 4: Узнайте ID группы

ID группы можно найти:
- В URL сообщества: `vk.com/club123456789` → ID = `123456789`
- Или в "Управление" → "Настройки" → внизу страницы

## Шаг 5: Настройте Long Poll API

1. Перейдите в "Управление" → "Настройки" → "Работа с API" → "Long Poll API"
2. Включите Long Poll API
3. Версия API: `5.199` (или последняя)
4. В "Типы событий" включите:
   - Входящее сообщение

## Готово!

При запуске `install.sh` вам понадобятся:
- **ID группы**: `123456789`
- **Токен группы**: `vk1.a.XXXXX...`
